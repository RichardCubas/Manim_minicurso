## Descripción

Plantilla base del minicurso de **Manim** desarrollado en la **Universidad Científica del Sur**.

Este repositorio reúne ejemplos y recursos para iniciar en la creación de animaciones educativas con **Manim** usando **Google Colab**. Incluye una estructura de trabajo lista para modificar y adaptar a distintos contextos académicos.

### Contenido

- Instalación y configuración en Google Colab
- Estructura básica de una escena en Manim
- Animación de textos y objetos
- Uso de coordenadas y posicionamiento
- Inserción de imágenes
- Transformaciones entre elementos
- Integración de narración en las animaciones

### Objetivo

Brindar una plantilla práctica que sirva como punto de partida para la elaboración de materiales visuales, presentaciones animadas y recursos didácticos con Manim.
